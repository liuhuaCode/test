# git introduce  

# step 
* 查看仓库状态  `git status`  
* 初始化仓库  `git init`  
* 首次提交代码 `git add -A`  
* 修改提交代码  `git add -u/[document name]`  
* 确认提交到本地仓库  `git commit -m "statement"`  
* 添加远程仓库 `git remote add  theName url`  
* 产看远程仓库 `git remote -v`  
* 提交代码到远程 `git push theName master`  
* 从远程仓库拉下代码 `git pull theName master`
